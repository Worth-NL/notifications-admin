{{ content_metadata( data={ "Last updated": "8 February 2023", "Next review due": "26 April 2023" } ) }}

The GOV.UK Notify roadmap shows what we’re working on and some of the things we’ve done.

The roadmap is only a guide. It does not cover everything we do, and some things may change.

You can contact us if you have any questions about the roadmap, want to get involved in user research or have suggestions for new features.

Things we’re working on now

  • Migrating our database and applications from GOV.UK PaaS to Amazon ECS
  • Let services create letters in the Welsh language to comply with the Welsh Language Act 1993
  • Add attachments to letters
  • Improve our email functionality so users can add images, QR codes and have more formatting options
  • Make it easier for services to request to go-live
  • Make it easier for users to join a live service
  • Design and test reusable content patterns for text messages
  • Integrate with our letter provider's API to improve the letter sending process
  • Improve our front-end code to meet our targets for accessibility
  • Migrate our database from GOV.UK PaaS to one hosted using Amazon RDS

Things we’ll do later

April to August 2023

  • Provide more detailed reporting and usage data for services
  • Provide reusable text message templates
  • Explore ways to let services send one-off messages
  • Continue to explore other innovations on the mobile and email communications market, and improvements to our product
  • Continue to improve our front-end code to meet our targets for accessibility
  • Migrate our applications from GOV.UK PaaS to Amazon ECS

Things we’ve done

  • Let services upload their own logo for email and letters
  • Let services choose their own two-factor authentication method
  • Let services receive text messages without having to contact the Notify team first
  • Made it easier to find information and guidance on the Notify website
  • Made it quicker and easier to turn off trial mode restrictions
  • Developed first version of reusable content patterns for text messages
  • Updated the version of the GOV.UK Design System that we use
