Guidance

This guidance is for teams using GOV.UK Notify to send emails, text messages and letters.

It explains how to:

  • edit and format messages
  • change your email branding
  • add letter branding
  • receive text messages
  • add a reply-to email address
  • schedule emails and text messages
  • send files by email
  • set permissions for team members
  • send messages using templates
  • change your text message sender
  • upload a letter

More information

The GOV.UK Service Manual has advice on:

  • planning and writing text messages and emails
  • writing effective letters
  • sending emails from your service domain
  • sending text messages securely
