Unsubscribe links

You must let people opt out of receiving subscription emails.

If recipients report your emails as spam we can stop you from using GOV.UK Notify.

Transactional emails do not need to include an unsubscribe link.

The GOV.​UK Service Manual explains the difference between transactional and subscription messages.

How to add unsubscribe links

Notify makes it easy to add:

  • a one-click unsubscribe button to the email header
  • an unsubscribe link at the bottom of your email

If you use the Notify website to send emails

  1. Go to the {{ service_link(current_service, 'main.choose_template', 'templates') }} page.
  2. Add a new email template or choose an existing email template.
  3. Select Allow recipients to unsubscribe.

This will add a button to the email header. Recipients can request to opt out of unwanted emails by clicking on the button.This will also add an unsubscribe link at the bottom of the email.

We will report any email unsubscribe requests on the {{ service_link(current_service, 'main.service_dashboard', 'dashboard') }}. You can download the report to manage your unsubscribe requests. This report is available for up to 90 days.

If you use the Notify API to send emails

You’ll need a developer on your team to set it up for you.

To add one-click unsubscribe, follow the instructions in our API documentation.
