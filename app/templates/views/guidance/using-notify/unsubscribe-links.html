Unsubscribe links

Subscription emails must include an unsubscribe link in the body of the message.

Transactional emails do not need to include an unsubscribe link.

How to add an unsubscribe link to an email

GOV.UK Notify uses Markdown to format links.

Use square brackets around the link text and round brackets around the URL. Make sure there are no spaces between the brackets or the link will not work. For example:

[Unsubscribe](https://www.gov.uk/example)

One-click unsubscribe

We are working on adding one-click unsubscribe to GOV.UK Notify. We expect to make this feature available by June 2024.

Your emails will still need to include unsubscribe links in the body of the message.
