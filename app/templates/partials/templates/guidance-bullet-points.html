Bullet points

Use bullet points to help words or phrases stand out in your emails and letters.

You can either:

  • add the same bullet points to every message you send
  • add personalised bullet points each time you send a message

1. Add the same bullet points to every message you send

Copy this example to add bullet points:


    Introduce bullet points with a lead-in line ending in a colon:

    * leave one empty line space after the lead-in line
    * use an asterisk or a dash followed by a space to add an item
    * start each item with a lowercase letter, do not end with a full stop
    * leave one empty line space after the last item

To create sub-items, add an indent of 2 spaces before the asterisk or dash.

2. Add personalised bullet points each time you send a message

Copy this example to add personalised bullet points each time you send a message:


    Introduce bullet points with a lead-in line ending in a colon:

    ((bullet points))

    Leave one empty line space before the next paragraph.

Your spreadsheet of recipient details should include a column for each of the bullet points.

Create a column with the placeholder name as a title for each bullet point your list will have. Then add one list item per column.

This is an example of a spreadsheet that will create personalised bullet points:


email address, bullet points, bullet points, bullet points
recipient-email@example.com, item 1, item 2, item 3
recipient2-email@example.com, item 1, item 2

When you send a message, Notify will replace the placeholder with the bullet points in your spreadsheet.
